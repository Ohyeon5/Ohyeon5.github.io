Google drive API from remote server

SCP to the local computer, then uploading to gdrive is often a big pain.
Instead, we can use the gdrive command line tool.
To install, follow the instructions in the gdrive.
Remember to follow the instructions for the remote server.
Then, you can upload files to the drive using the following commands.

First, find the drive id of the folder you want to upload to.
gdrive drives list
Then, upload the file using the following commands.
gdrive files upload --parent [drive id] [filename]
gdrive files upload --recursive [foldername] --parent [drive id]

To specify the directory of the file, [drive id] can be replacd with the directory's URL's last part.
